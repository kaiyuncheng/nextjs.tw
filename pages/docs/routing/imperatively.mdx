# 透過程式切換頁面

[`next/link`](/docs/api-reference/next/link.md) 在大部分情況下能夠滿足您的需求，不過他不是唯一的解決方案。[`next/router` 的文件](/docs/api-reference/next/router.md) 值得您一看。

下方的範例展示了如何使用 [`useRouter`](/docs/api-reference/next/router.md#useRouter) 來達到基本的頁面跳轉：

```jsx
import { useRouter } from 'next/router'

export default function ReadMore() {
  const router = useRouter()

  return (
    <button onClick={() => router.push('/about')}>
      Click here to read more
    </button>
  )
}
```
