#### 客戶端和伺服器
在 Web 應用程式中，**客戶端**是指使用者設備上的瀏覽器，它向伺服器發送請求以獲取您的程式碼。然後將伺服器接收到的回應轉換為用戶可以與之互動的界面。

![客戶端和伺服器](https://nextjs.org/static/images/learn/foundations/client-server.png "客戶端和伺服器")

**伺服器**是指資料中心中的電腦，它可以儲存您的程式碼、接收來自客戶端的請求、進行一些計算後傳送回適當的回應。
