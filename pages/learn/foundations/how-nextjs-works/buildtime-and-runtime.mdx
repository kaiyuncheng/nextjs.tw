#### 建置時期和執行時期
**建置時期（ build time ）** 是為正式環境準備程式碼的一系列步驟的名稱。

當您構建應用程式時，Next.js 會將您的程式碼轉換為用於正式環境的優化檔案，準備好部署到伺服器並供使用者使用。這些檔案包括：

+  靜態生成頁面的 HTML 檔案
+  用於在伺服器上渲染頁面的 JavaScript 程式碼
+  用於在客戶端上使頁面具有互動性的 JavaScript 程式碼
+  CSS 檔案

**執行時期（ runtime ）** 是指在建置和部署應用程式之後，應用程式處理使用者請求並給予對應回應的時間段。
